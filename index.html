Java Interface – सभी नियम

  • 1. Interface बनाने के लिए interface keyword का उपयोग होता है।
  • 2. Interface में methods by default public और abstract होते हैं।
  • 3. Interface में method का body नहीं होता (Java 7 तक)।
  • 4. Java 8 से default methods allowed हैं।
  • 5. Java 8 से static methods भी allowed हैं।
  • 6. Interface के variables by default public static final होते हैं।
  • 7. Interface का object नहीं बनाया जा सकता।
  • 8. Class interface को implements keyword से implement करती है।
  • 9. Implement करने वाली class को interface के सभी methods override करने पड़ते हैं।
  • 10. एक class एक से ज्यादा interfaces implement कर सकती है (Multiple Inheritance support)।
  • 11. Interface, दूसरे interface को extends कर सकता है।
  • 12. Interface में constructor नहीं होता।
  • 13. Interface में main() method हो सकता है।
  • 14. Java 9 से interface में private methods allowed हैं।
  • 15. Interface reference का उपयोग करना best practice माना जाता है।

निष्कर्ष

Interface का उपयोग abstraction, loose coupling और multiple inheritance के लिए किया जाता है।
